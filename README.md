# 🧠 A Deep Learning Approach for Brain Tumor Segmentation using U-Net in MRI Images

A deep learning-based medical image segmentation project that uses the **U-Net architecture** to automatically detect and segment brain tumors from MRI scans. This project is trained on the **BraTS2020 dataset** and focuses on accurate tumor localization using semantic segmentation.

## 📌 Overview

Brain tumor segmentation is a critical task in medical imaging that helps doctors diagnose, monitor, and plan treatment effectively. Manual segmentation is time-consuming and depends on expert knowledge.

This project automates the process using **Convolutional Neural Networks (CNNs)** and a custom **U-Net model** to segment tumors from MRI images.

---

## 🎯 Objectives

- Detect tumor regions from MRI scans  
- Segment tumors with pixel-level precision  
- Reduce manual effort in diagnosis  
- Build a reliable AI-based healthcare solution  

---

## 🧬 Dataset Used

## 📂 BraTS2020 Dataset

The **Brain Tumor Segmentation (BraTS) 2020** dataset contains MRI scans of glioma patients with expert-labeled masks.

### MRI Modalities:

- T1
- T1ce (used in this project)
- T2
- FLAIR

### Dataset Preparation:

- Selected **50 patients**
- Extracted **3,336 MRI slices**
- Removed empty slices (without tumor)
- Resized images from **240×240 → 128×128**
- Normalized pixel values between **0 and 1**

## 🏗️ Model Architecture

## U-Net Architecture

U-Net is widely used for biomedical image segmentation.

### Main Components:

### 🔹 Encoder
- Convolution layers
- MaxPooling
- Feature extraction

### 🔹 Bottleneck
- Learns deep spatial features

### 🔹 Decoder
- Upsampling
- Skip connections
- Tumor reconstruction mask

### 🔹 Output Layer
- Sigmoid activation
- Binary segmentation mask

## 📈 Training Details

| Parameter | Value |
|----------|-------|
| Optimizer | Adam |
| Learning Rate | 0.0001 |
| Epochs | 25 |
| Batch Size | 8 |
| Loss Function | Binary Crossentropy + Dice Loss |

---

## 📊 Performance Results

| Metric | Score |
|-------|------|
| Training Dice Score | 0.559 |
| Validation Dice Score | **0.672** |
| Validation Loss | **0.465** |

### ✅ Key Result:

The model achieved a **Dice Score of 0.672**, showing strong overlap between predicted and actual tumor masks.
